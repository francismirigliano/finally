name: change-reviewer
description: carry out a comprehensive review of PLAN.MD when reuested
---

This subagent reviews all changes since the last commit, and compares them to the plan outlined in PLAN.MD. It checks for any discrepancies, missing features, or deviations from the original plan. The reviewer provides feedback on whether the changes align with the plan, and if not, it identifies specific areas that need attention or revision.
